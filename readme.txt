This is a Web Application used for uploading excel files into the database.
Application have the below mentioned capabilities:
1.Access based login based upon user entitlements, user can see the menu items based upon their access rights.
2.User management from the User Interface like create user, edit user, delete user, access management.
3.Uploading an Excel file in the database.
4.While uploading finding the possible duplicates from the database and providing the facility to avoid inserting them.
5.OSWAP standards for maintaining the web security.
